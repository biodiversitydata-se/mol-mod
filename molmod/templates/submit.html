How to submit data to the ASV database and Bioatlas


Some intro about relationship between ENA-ASV DB-Bioatlas, perhaps referring to About page.


Overview of planned flow, to be described.

  1. Data contributor (DC) deposits raw data in ENA or other publicly available repository
  2. DC downloads Excel template from module (this page)
  3. DC creates Bioatlas account (Login-page)
  4. DC logs in (Login-page)
  5. DC requests upload permission (File upload-page)
  6. DC uploads Excel or TSV-files (File upload-page)
  7. We import data into ASV database, and apply standard taxonomic re-annotation.
  8. We create a new data resource in the Integrated Publishing Toolkit (IPT), and add new IPT-user, if needed.
  9. We email DC with IPT-login-info and 'Databas√§gareavtal' contract
  10. DC returns signed contract, supplies metadata and publishes data in IPT
  11. Bioatlas admin lets us know when data have been imported into Bioatlas
  12. We flag dataset as published, making it searchable in webapp
  13. We regularly update taxonomic annotation, and publish new dataset versions in IPT.
