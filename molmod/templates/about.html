About the Swedish ASV database


The database of Swedish Amplicon Sequence Variants (ASVs) is an interface to sequence-based observations in the Bioatlas, which is part of the Swedish Biodiversity Data Infrastructure (SBDI). Read more about Molecular biodiversity data in SBDI here.
